# 学生画像介绍

这是一个用于 AI 教程和课程设计的标准学生画像示例。在设计课程或生成教学内容时，提供清晰的学生画像可以帮助 AI 更好地调整内容的深度、术语和教学节奏。

## 1. 学生身份 (Student Profile)
* **教育背景**：计算机科学或相关专业的大二/大三学生。
* **学习目标**：希望从基础的理论学习过渡到实际的工程应用，能够独立完成包含前后端和 AI 交互的全链路项目。
* **学习风格**：实践导向（喜欢边写代码边学习），对过多的纯理论推导容易失去耐心，更喜欢通过具体的例子、图解和代码片段来理解抽象概念。

## 2. 已掌握知识与课程 (Prerequisites & Current Skills)
* **编程语言**：熟练掌握 Python（了解基本语法、面向对象编程，能使用常用内置库）；了解基础的 JavaScript/HTML/CSS。
* **数学基础**：已修完《高等数学》、《线性代数》和《概率论与数理统计》（掌握基础概念，但不擅长复杂的数学公式手推）。
* **计算机基础**：掌握基本的数据结构（数组、链表、树、哈希表）和算法；了解基础的计算机网络概念（HTTP 协议基础）。
* **开发工具**：熟悉 Git 的基本提交和分支操作，能熟练使用 VS Code 及常用插件。

## 3. 兴趣点与探索方向 (Points of Interest)
* **大模型应用开发**：对如何将大型语言模型（LLM）集成到实际应用中（例如开发智能客服、基于自有文档的 RAG 问答系统）非常感兴趣。
* **效率与自动化工具**：喜欢编写脚本来自动化日常繁琐任务，对提高生产力的 AI 工具和工作流充满热情。
* **前沿技术落地**：对 Agent（智能体）的工作原理及其在软件工程、数据分析等垂直领域的实际落地场景有强烈的探索欲。
