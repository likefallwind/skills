# 课程情况介绍

![Course Cover](./assets/course_cover.png)

## 课程概况

课程名称：Vibe Coding 101

### 课程目标

帮助非计算机专业的学生通过vibe coding学习AI。
注意，这门课既要入门，也要有一定的深入，我希望通过这门课的学习，学生能够成为一个相对有能力的vibe coder。
一些重要知识点，包括：
* vibe coding基本使用
* vibe coding IDE
* vibe coding与代码管理
* vibe coding in Terminal
* vibe coding常见工具：command, MCP, SKILLS等
这些只是一些启发，你要根据课程实际情况决定

### 授课形式

讲义+项目实战，注意项目实战更为关键

### 课程时长

AI根据需要自己决定



## 学生背景

- 技术背景：非计算机方向本科生，包括历史、金融、生物、物理等各个专业
- 学习目标：对 AI 有浓厚的兴趣，希望能够利用vibe coding解决实际问题
- 课时数：AI根据需要自己决定
- 前置经验：几乎零基础，有一点点python基础
